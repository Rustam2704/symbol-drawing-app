# Archive

Files moved here are not used by the running Drawing app. They are preserved for manual review instead of being deleted.

- `images/generated-practice-old/` — older generated practice image set.
- `images/old/` — legacy image variants.
- `images/sens-pen.png` — unreferenced standalone image.
- `assets/` — legacy sound effects that are no longer referenced by the application.
- `maintenance/codex-migration-recovery/` — one-time Codex workspace recovery notes and script.
- `vendor-source/perfect-freehand/` — upstream source checkout; the app uses the built runtime under `vendor/perfect-freehand-package/`.
